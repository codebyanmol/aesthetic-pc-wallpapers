## How to Use

1. Download or clone the repository

2. Extract the wallpapers if needed

3. Set the wallpaper using your desktop environment

Example for Linux users:

* Hyprland
* KDE Plasma
* GNOME
* XFCE
* i3 / bspwm

## Tips for Best Look

To get the best aesthetic desktop:

* Use a minimal icon theme
* Choose a clean font
* Match your terminal colors with the wallpaper
* Keep the desktop uncluttered

Simple setups often look the best.

## Contributing

If you want to contribute wallpapers:

1. Make sure the image is **high resolution**
2. Avoid heavy compression
3. Keep the **aesthetic / minimal style**
4. Open a pull request
